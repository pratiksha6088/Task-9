## CSS Laundry Services Button Interactive (Transforms)

## Task Title
CSS Laundry Services Button Interactive

## Description:
This project is designed to create an interactive Call-To-Action (CTA) button for a Laundry Service website. The button uses CSS Transform properties to make the webpage more attractive and interactive.

The main purpose of this task is to improve user experience by adding hover effects on the CTA button.

## Features

✔ Button increases in size on hover  
✔ Button tilts when mouse hovers over it  
✔ Smooth transition animation  
✔ Responsive layout  
✔ Simple and clean design  

## Technologies Used

- HTML
- CSS

## CSS Properties Used

### Transform
Used to modify the appearance of the button.

Examples:

- `scale()` → Increases button size
- `rotate()` → Tilts the button

Example:

```css
transform: scale(1.2) rotate(-8deg);
```

### Transition
Used to make animations smooth.

Example:

```css
transition: all 0.4s ease;
```
